# Math4 Dyslexia App

This project is designed to help individuals with dyslexia improve their math skills. The app will provide interactive lessons, exercises, and various tools to assist users in their learning journey.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SasukeUchiha667/math4-dyslexia-app.git
   ```

2. Navigate into the directory:
   ```bash
   cd math4-dyslexia-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```
   
## Usage

To start the app, run:
```bash
npm start
```
